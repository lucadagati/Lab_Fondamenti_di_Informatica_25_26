# Appendice (non documento principale)

Il documento **di riferimento** del Lab 2 è **`README.md`** in questa cartella: contiene istruzioni di installazione, workflow agentico e prompt pronti all’uso.

Questo file serve solo come **appendice** per:
- mappa dei livelli (0 -> 5),
- descrizione sintetica delle finalità pedagogiche,
- elenco dei materiali.

---

## Mappa livelli

- **Livello 0** (`esercizi/`): script singolo file, completamento, refactoring, debug.
- **Livello 1-2** (`progetti_piccoli/`): progettini (menu, parser da file) e modulo multi-file (`spese_modulo/`).
- **Livello 3-5** (`jupyter/`): notebook, celle Markdown+code, analisi guidata con dataset in `data/`.

---

## Finalità didattica

In ogni livello l’assistente in modalità agentica deve essere usato come “tutor operativo”:
- propone un piano,
- implementa con vincoli,
- tu esegui/verifichi,
- se serve fai debug con l’errore osservato.
